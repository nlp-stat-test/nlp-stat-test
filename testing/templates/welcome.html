About

This website is a demonstration of "NLPStatTest: A Toolkit for Comparing NLP System Performance", by Haotian Zhu, Denise Mak, Jesse Gioannini, and Fei Xia. A video tutorial is availble here. Please consult the manual for instructions and tips. You can find the source code on our GitHub repository under the terms of the MIT License. If you have not yet collect data, start with prospective power analysis, otherwise, you can upload your data or restore a previous session.
