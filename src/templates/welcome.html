About

This website is a demonstration of "NLPStatTest: A Toolkit for Comparing NLP System Performance", by Haotian Zhu, Denise Mak, Jesse Gioannini, and Fei Xia. They can be contacted for help or feedback.

While statistical significance testing has been commonly used to compare NLP system performance, a small p-value alone is not sufficient because statistical significance is different from practical significance. To measure practical significance, we recommend estimating and reporting of effect size. It is also necessary to conduct power analysis to ensure that the test corpus is large enough to achieve a desirable power level. We propose a three-stage procedure for comparing NLP system performance, and provide a toolkit, NLPStatTest, to automate the testing stage of the procedure. For future work, we will extend this work to hypothesis testing with multiple datasets or multiple metrics.

  • To get familiar with the toolkit, start with a 3-minute demo.
  • More info about the toolkit can be found in this paper and the manual.
  • To determine the minimal size of the test corpus with sufficient power, use prospective power analysis.
  • To compare system performance, click here to get started.
  • If you want to run the toolkit on your own machine (as a command line interface, or as a local GUI), you can download an executable and source code from the Releases page.
  • Our paper contains a list of references.

Data Policy

The web version of NLPStatTest uses the data file you upload (two columns of numbers) to calculate your results. We also use cookies. You can delete the most recent file you uploaded and clear the cookies by clicking the delete data link here or at the end of the interface. Otherwise, we will clear the data from time to time. We do not use uploaded data for any other purpose than to calculate your results.
